# Wildlife Motion Detection System

## Overview

This project uses a Raspberry Pi and a PIR motion sensor to detect movement, activate an LED, log the event to a CSV file and automatically generate a PDF report.

## Features

- Motion detection
- LED indicator
- CSV logging
- Automatic PDF reports

## Hardware

- Raspberry Pi 3 B+
- PIR Motion Sensor
- LED
- Breadboard
- Jumper wires

## Software

- Python
- ReportLab
- gpiozero
